Q-CLI 集成器

Universal CLI Bridge — 在浏览器中运行任意 CLI

🚀 快速开始

1
选择 CLI 点击左侧 CLIs 列表中的任意工具
2
开始使用 终端自动启动，直接输入命令操作
3
随时切换 点击其他 CLI 即可切换，运行状态互不干扰

⭐ 特色功能

⌨️ 快捷键

Ctrl+K→打开命令面板
Ctrl+B→切换侧边栏
Ctrl+Shift+F→终端内搜索
Ctrl+L→清屏
Ctrl+I→AI 对话
Ctrl+= / Ctrl+-→调整字体大小
Ctrl+Shift+C→复制选中文本
Ctrl+Shift+V→粘贴

💡 使用技巧

  • 使用 搜索框 快速过滤 CLI 列表
  • 创建 文件夹 分组管理你的 CLI 工具
  • 拖拽文件到终端区域可快速 上传文件
  • 点击 ⟳ 按钮从 PATH 自动发现新 CLI
